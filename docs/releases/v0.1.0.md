# YouTube Digest v0.1.0

This is the first usable project snapshot of YouTube Digest: a productized
derivative of `zarazhangrui/youtube-to-ebook`, now shaped around a stable CLI,
auditable artifacts, and real scheduled use through agents or VPS automation.

## Highlights

- Added the `youtube_digest` Python package and `youtube-digest` CLI.
- Added channel discovery through YouTube Data API and explicit single-video
  processing with `generate --video-url`.
- Added Supadata transcript fetching, defaulting to native transcripts.
- Added pluggable writer support for Anthropic, Gemini, DeepSeek, OpenAI,
  OpenRouter, and generic OpenAI-compatible APIs.
- Added magazine mode with analysis, draft, optional expansion, and saved
  intermediate artifacts.
- Added EPUB generation, HTML archive output, and readable artifact filenames.
- Added SQLite processed-video tracking for idempotent scheduled runs.
- Added structured JSON output with legacy `errors` plus machine-readable
  `error_details`.
- Added explicit `--reuse-transcript` and `--reuse-analysis` for cost-controlled
  reruns.
- Added Docker, systemd, CI, smoke-test, deployment docs, and a thin Hermes skill
  entry point.

## Validation

- CI on `main` passed before release.
- `scripts/quality.sh` passes locally with 36 tests.
- Real single-video generation and artifact reuse have been verified manually.

## Notes

The repository preserves upstream history and attribution. This release is meant
as a practical baseline for real personal use: running scheduled digests on a VPS,
tracking selected YouTube channels, and iterating from actual failures and needs.
